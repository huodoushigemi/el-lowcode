## Feature

- [x] 支持 `Form` `H5` `BI` 开发
- [x] 支持表单导出 `JSON Schema`
- [x] `CRUD` 生成器
- [x] 支持指令 `v-for` `v-if` `v-model` `v-slot` `v-html`
- [x] 支持任意一处使用 `JavaScript` 编写自定义代码
- [x] 支持远程插件扩展
- [x] 提供在线示例模板
- [x] 生成静态代码
- [x] 支持导出 `pdf` `png` `svg`
- [x] 数据源
- [x] AI 辅助：图转码
- [ ] 支持 `uniapp` `小程序`
- [ ] 多主题切换
- [ ] Excel 智能解析
- [ ] 更多功能等你探究……
